## Features:

- Login and Registration
- Notification
- Private Messaging
- Post CRUD functionality
- Comment feature
- Profile Customization
- Followers/Following feature
- Search Feature

## Technology Stack:

- TypeScript
- Node js
- Express Js
- MongoDB
- React
- Redux

## Usage

### Env Variables

Create a .env file in then root and add the following

```
NODE_ENV = development
PORT = 5000
MONGO_URI = your mongodb uri
```

## Install Dependencies

```
npm install
cd frontend
npm install
```

### Run

```
# Run frontend
npm run client
# Run backend
npm run server
```
